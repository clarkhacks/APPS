Request Maker 1.0

← Go Back

Request Maker is a tool for making Restful requests. There are bugs, errors, and what-nots. This is not a polished product (as of yet).

Use

Select your request type (PUT, GET, POST, PATCH).
Enter your API url (must be https)
Fill in parameters.

For each key there must be a value.
For the parameter you leave unused keep their fields blank.
